## How I build

I keep things simple. One VPS, Docker Compose, Traefik as reverse proxy, CI/CD with GitHub Actions. No Kubernetes, no over-engineering. Ship fast, iterate, fix what breaks.

Most of my repos are private — I build products, not portfolios. But [zerionis-template-app](https://github.com/GaboMiranda/zerionis-template-app) shows how I structure deploys if you're curious.
